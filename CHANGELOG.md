# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-30

### Added
- Core monorepo and Docker Compose files.
- Flyway database migration scripts from schema structure to symbol seeds.
- JPA entities, repository interfaces, global DTO records, MapStruct mappers, and JWT filters.
- Core business REST controllers for Auth, Projects, and Diagram History versioning.
- AI Graph Generation Pipeline services (cycle checkers, active path propagators, level BFS layout computation).
- Spring AI OpenAI orchestration pipeline and Server-Sent Events (SSE) streaming progress.
- React 19 Frontend Web Client with interactive canvas nodes, stopper state toggling, and local path highlighting.
- Default example projects for Demo Mode: Industrial Cooling Water, Electrical Distribution, and Hydraulic Press.
